# bockzee

A minimalist anonymous compliment board built with Expo and React Native. It lets people post short messages, see a live feed, and browse a lightweight mobile app shell.

## Running locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the Expo dev server:
   ```bash
   npm start
   ```
3. Use the Expo CLI prompts to run on Android, iOS, or web.

## TypeScript

The main app entry now lives in `App.tsx` and is type-checked with the repository's TypeScript configuration.
